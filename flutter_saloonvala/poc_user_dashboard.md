# Proof of Concept: User Dashboard in Flutter

## Overview
This POC demonstrates converting the Android User Dashboard to Flutter.

## What Will Be Converted

### UI Components
- Welcome section
- Search bar
- Category grid
- Trending services horizontal list
- Quick actions grid
- Offers carousel
- My bookings horizontal list
- Nearby salons horizontal list

### Functionality
- User profile display
- Search suggestions
- Category navigation
- Salon filtering
- Location-based salon listing

## Implementation Plan

### Step 1: Project Setup
```bash
flutter create saloonvala_flutter
cd saloonvala_flutter
```

### Step 2: Add Dependencies
Update `pubspec.yaml` with required packages

### Step 3: Create Models
- User model
- Salon model
- Category model
- Booking model

### Step 4: Create Services
- API service (using dio)
- Storage service (using shared_preferences)
- Location service (using geolocator)

### Step 5: Create Providers
- User provider
- Salon provider
- Location provider

### Step 6: Create UI
- Dashboard screen
- Reusable widgets (cards, lists)
